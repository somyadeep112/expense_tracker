# expense_tracker

# 💰 Expense Tracker App

A full-stack web application to manage daily expenses with budget tracking and reports.

---

## 🚀 Features

- 🔐 User Authentication (Login/Register)
- ➕ Add / Edit / Delete Expenses
- 📊 Dashboard with charts
- 💰 Monthly Budget Tracking
- ⚠️ Budget Exceed Warning
- 📈 Reports (Category & Monthly)
- 📁 Export to CSV
- 📱 Responsive UI

---

## 🛠️ Tech Stack

- Backend: Flask (Python)
- Database: SQLite
- Frontend: HTML, CSS
- Charts: Chart.js

---

## ▶️ How to Run

```bash
pip install flask flask_sqlalchemy werkzeug
python app.py
